# 世界书助手脚本

该脚本用于在酒馆内更高效地查看和处理世界书，重点覆盖：

- 世界书提取：列表、全局/角色/聊天绑定、激活记录监控
- 世界书处理：导入归一化、批量查找替换、批量启停、排序与标准化
- 世界书编辑：可视化条目列表 + 详细字段编辑
- 世界书保存：直接保存到当前世界书、快照回滚、导出

## 使用方式

1. 在酒馆助手中导入打包后的 `dist/worldbook_assistant_build/index.js`。
2. 启用脚本后，点击左下角魔法棒，菜单中会出现 `世界书助手`。
3. 点击该菜单项打开/关闭面板（面板可拖动，条目区支持滚动）。

## 导入策略

- `覆盖当前草稿`：替换当前面板中的条目（未立即写盘，需再点保存）
- `追加到当前草稿`：把导入条目追加到当前草稿
- `导入为新世界书`：直接新建/覆盖目标世界书

如果 JSON 结构解析失败，脚本会提供“按酒馆原生方式导入”的兜底选项。
